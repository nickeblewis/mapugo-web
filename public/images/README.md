# `/public/images`

Drop static images here. They are served from the root of the site, so
`public/images/hero.jpg` becomes `https://…/images/hero.jpg`.

Expected files (referenced from `app/components/Hero.tsx`):

- `hero.jpg` – background for the homepage hero section.

If you want the exact look of the headforcode-2026 site, copy
`../../../headforcode-2026/public/images/head_for_code_cover.jpeg` to
`public/images/hero.jpg` in this project.
